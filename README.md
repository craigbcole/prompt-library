# Grok Imagine Prompt Library

A free, private Chrome/Brave extension that automatically saves every prompt and generation from Grok Imagine.

### Features
- Click any generated image → “Save to Prompt Library”
- Beautiful searchable grid with thumbnails
- Click any card to see full prompt
- One-click **Regenerate** and **Share** (copies prompt to clipboard)
- Export entire library as JSON
- Clear library button (for privacy)

### How to Install
1. Download the latest release (or clone this repo).
2. Go to `chrome://extensions/` (or `brave://extensions/`)
3. Enable **Developer mode** (top right)
4. Click **“Load unpacked”** and select the folder.

All data stays 100% on your computer — nothing is sent anywhere.
